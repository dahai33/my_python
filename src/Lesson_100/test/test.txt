



cmd + [  移动到光标之前

cmd + ]  移动到光标之后

fn + delete  反方向删除

shift + tab  取消缩进

cmd + 鼠标放在代码上  显示所选部分简要信息

cmd + /  注释/取消注释

cmd + op + L  格式化代码

cmd + D  复制当前行并粘贴至下一行

fn + op + delete  删除到单词的末尾

op + delete  删除到单词的开头

shift  + shift  查询任何东西

cmd + F  文件内查找

cmd + shift + F  全局查找

cmd + R  文件内替换

cmd + shift + R  全局替换

cmd + Y  删除一整行
